# Claude Code Interactive Learning Skill

Welcome to the Claude Code Interactive Learning Skill! This skill provides a guided, step-by-step learning experience for Claude Code, from beginner basics to advanced parallel development techniques.

## Installation

To install this learning skill:

```bash
# Install the skill using Claude Code
claude-code install claude-code-tutorial
```

Or if using the CLI:

```bash
# Install via npm
npm install -g claude-code-tutorial
```

## Usage

Once installed, you can start the interactive learning experience using:

```bash
/learn-claude-code
```

This will start a guided tutorial that covers:

1. **Beginner Level**: Basic concepts and setup
2. **Intermediate Level**: Working with tasks and constraints
3. **Advanced Level**: Parallel development with worktrees
4. **Expert Level**: Advanced patterns and best practices
