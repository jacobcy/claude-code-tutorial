# Search Claude Code Help

Search for documentation by keyword across all available sources.

## Usage

```
/helper search <keyword>
```

## Search Sources

- Local knowledge base
- Current project files
- Official documentation
- Web resources (fallback)

## Examples

```
/helper search worktree
/helper search task management
/helper search testing coverage
```
