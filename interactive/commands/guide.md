# Claude Code Learning Guides

Start a scenario-based learning path for mastering Claude Code.

## Usage

```
/helper guide <scenario>
```

## Available Scenarios

### new-project
Starting fresh with Claude Code:
- Setting up TASK.md
- Configuring AGENT_RULES.md
- Best practices for first-time users

### debug
Troubleshooting and recovery:
- Error reporting techniques
- Unstuck strategies
- File access issues

### team
Team collaboration patterns:
- Shared TASK.md conventions
- Feature branch workflows
- Code review with AI-generated code

### deploy
Pre-release and deployment:
- Review checklist
- Security review
- Deployment documentation

### parallel
Worktree-based parallel development:
- Git worktree fundamentals
- Isolated development environments
- Concurrent feature development

## Examples

```
/helper guide new-project
/helper guide debug
/helper guide team
```
