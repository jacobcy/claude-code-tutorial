# Claude Code Tutorial Skills Manifest

This document lists all the available skills for learning Claude Code.

## Available Skills

### 1. `/learn-claude-code`
- **Description**: Main interactive tutorial entry for Claude Code
- **Usage**: `/learn-claude-code`
- **Purpose**: Provides an interactive introduction to Claude Code, covering basic concepts, navigation, and essential commands

### 2. `/learn-claude-advanced`
- **Description**: Access advanced Claude Code topics and techniques
- **Usage**: `/learn-claude-advanced`
- **Purpose**: Provides access to advanced Claude Code topics, including complex workflows, debugging strategies, and power features

### 3. `/learn-claude-code start`
- **Description**: Beginner tutorial for Claude Code
- **Usage**: `/learn-claude-code start`
- **Purpose**: Starts a guided tutorial for beginners, introducing core Claude Code concepts and basic usage patterns

### 4. `/learn-claude-code worktrees`
- **Description**: Learn about worktrees and parallel development in Claude Code
- **Usage**: `/learn-claude-code worktrees`
- **Purpose**: Provides instruction on using Git worktrees with Claude Code for parallel development, branch management, and isolated feature work

### 5. `/learn-claude-code safety`
- **Description**: Learn about safety practices in Claude Code
- **Usage**: `/learn-claude-code safety`
- **Purpose**: Provides guidance on safe coding practices, security considerations, and protective measures when using Claude Code for development

## How to Use

Simply type any of the above commands in Claude to launch the corresponding tutorial or learning module. Each skill is designed to provide targeted education on different aspects of Claude Code usage.
